# Özkan Oto Boyasız Göçük Düzeltme

İstanbul Gaziosmanpaşa'da 2013 yılından beri hizmet veren Özkan Oto'nun kurumsal web sitesi. Boyasız göçük düzeltme (PDR), kaporta onarımı ve dolu hasarı konusunda uzman ekibimizle aracınızın orijinal değerini koruyoruz.

## Kurulum

```bash
npm install
npm run dev
```

Geliştirme sunucusu `http://localhost:3000` adresinde başlatılır.

## Production Build

```bash
npm run build
npm start
```

## Teknolojiler

Next.js 14, TypeScript, Tailwind CSS, Framer Motion

## Özellikler

- Responsive tasarım
- 53 görselli galeri
- 31 gerçek müşteri yorumu
- WhatsApp entegrasyonu
- Google Analytics desteği
- SEO optimize
